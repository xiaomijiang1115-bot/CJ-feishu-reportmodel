## 周报格式

### 周报内容格式
周报与日报格式完全一致：
- Heading2：周次（如`2026年 第18周（5月3日-5月9日）`）
- Heading3：行业（如`一、商业航天`）
- Bullet：事件内容

### 周报特有 - 投资观点（末尾）
```
💡 投资观点
机会
商业航天：...
低空经济：...
...
风险
...
报告完成时间：2026年5月9日 | 数据来源：Tavily搜索
```

## 通用内容规范

### 来源格式
- `媒体名 月日` 或 `媒体名 时间`
- 多媒体用 `/` 分隔：`证券时报/财联社`

### 标注符号
| 符号 | 含义 |
|------|------|
| 💡 | 重要洞察、正面信息 |
| ⚠️ | 风险警示、负面信息 |

### 多事件分隔
同一行业多条内容用 `\n` 分隔：
```
事件1 | 来源 | 💡标注
事件2 | 来源 | ⚠️标注
```

## 典型任务流程

### 1. 更新日报/及时事件预警
1. 用 `read` 找到该日期的block结构
2. 用 `update_block` 替换"暂无特定新闻"为真实内容
3. 格式：`事件 | 来源 | 💡/⚠️`

### 2. 插入新日期到日报/及时事件预警
1. 确认分割线位置
2. 用 `insert` 在分割线后插入
3. 格式：Heading2日期 + 8个Heading3行业 + 各行业Bullet内容

### 3. 更新周报
1. 确认分割线位置
2. 用 `insert` 在分割线后插入新周次
3. 格式：Heading2周次 + 8个Heading3行业 + 各行业Bullet内容
4. 追加投资观点部分（机会+风险）

## 文档协同规则（重要！）

### 日报与及时事件预警协同
**每次更新日报前，必须先读取及时事件预警（QtlvdTFW9oCdHSxAtGPcQIYRnGb）进行交叉参考**：
1. **内容一致性**：同一事件在日报和预警中保持描述一致
2. **去重检查**：避免重复录入同一事件到日报
3. **时间协同**：预警中的重要事件及时同步到日报对应行业

### 具体操作
1. 更新日报前，先 `read` 及时事件预警的最新日期内容
2. 对照预警内容，确保日报不遗漏重要事件
3. 同一事件在两文档中保持来源和标注一致

## 注意事项
- 每条内容尽量包含量化指标（%、万台、亿元等）
- 延续报道标记：`[续·X月X日]`
- 重大事件必须同时体现在日报和预警中
